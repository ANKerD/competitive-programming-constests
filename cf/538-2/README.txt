como usar os scripts:

Para C/C++:
escreva seu código fonte no arquigo <code.cpp>
invoque o comando ./cpp <arquivo_de_entrada>
substitua arquivo_de_entrada por um arquivo contendo a enrada do seu programa.
crie um arquivo com o mesmo nome com um "x" apendido no final
e ponha nele a saída esperada para o caso de teste.

Exemplo:
./cpp.sh 1
.________________________________________________________________|
|																 |
|Hoje eu commmmi 8 bolinhos de arroz							 |
|-------------													 |
|Hoje eu comi 8 bolinhos de arroz								 |
|																 |
|________________________________________________________________|

// Compila e executa o arquivo `code.cpp`

// Joga o texto no arquivo "1" para a entrada do programa

// Imprime um separador

// Joga o texto no arquivo "1x" para a saída padrão
// abaixo da saída do seu código.

Sobre python:
escreva seu código fonte no arquigo <code.py>

a mesma estrutura também é válida para python. Basta executar o caso de
teste chamando o utilitário ./py.sh
